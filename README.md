# Template Dashboard - Smartcoop Broadcast

Dashboard template dengan Vue 2.6.11 dan Bootstrap Vue 2.x

## Teknologi

- Vue 2.6.11
- Bootstrap Vue 2.21.2
- Bootstrap CSS 4.6.2
- ApexCharts
- jQuery
- Select2
- SweetAlert2
- Toastr

## Instalasi

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run serve
```

3. Build untuk production:
```bash
npm run build
```

## Fitur

- Dashboard dengan statistik
- Sidebar navigation
- Notifikasi dropdown
- User profile dropdown
- Dark mode toggle
- Responsive design
- ApexCharts integration

## Custom CSS

File `src/assets/css/custom.css` berisi custom styling yang menggantikan Tailwind CSS dengan Bootstrap-compatible styles.
